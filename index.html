📁 How to Save Forms in a Folder

1
Open Chrome Settings

Click the three dots (⋮) in the top right corner of Chrome, then click Settings.

2
Go to Downloads

On the left sidebar, click Downloads.

3
Turn on "Ask where to save"

Toggle ON the option that says "Ask where to save each file before downloading".

4
Create your folder (once only!)

On your Desktop, right click → New Folder → name it "Sarwari Client Information Forms".

5
Save the form

Click the 💾 Save button → browser will ask where to save → select your "Sarwari Client Information Forms" folder → Done! ✓

💡 Tip: The file will be automatically named after the client and their event date — e.g. Ahmed_2025-08-15.json
